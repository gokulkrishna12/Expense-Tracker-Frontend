# 💰 Expense Tracker Frontend

A modern and responsive **Expense Tracker Frontend** built using **React.js**, **Vite**, and **Bootstrap**. This application helps users securely manage their daily expenses with JWT Authentication and a beautiful glassmorphism UI.

---

## 🚀 Features

- 🔐 User Login & Registration
- 💰 Add Expense
- ✏️ Update Expense
- 🗑️ Delete Expense
- 📋 Expense List
- 📊 Expense Category Pie Chart
- 📈 Dashboard Statistics
- 🔍 Search Expenses
- 🗂️ Filter by Category
- 📄 Export Expenses to CSV
- 🌙 Dark / ☀️ Light Theme
- 👁️ Show / Hide Password
- 💾 Remember Email
- 📱 Fully Responsive Design
- ✨ Glassmorphism UI

---

## ▶️ Run Project

Clone Repository

```bash
git clone https://github.com/yourusername/expense-tracker-frontend.git
```

Go to project

```bash
cd expense-tracker-frontend
```

Install Dependencies

```bash
npm install
```

Run Project

```bash
npm run dev
```
